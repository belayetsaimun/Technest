Keyboard Shortcuts

Use these keyboard shortcuts to quickly control the theme:

Ctrl + Shift + T Toggle between light and dark mode
Ctrl + Shift + V Show/hide theme variant selector

How to Use

  1. Quick Toggle: Click the theme toggle button in the top-right corner
  2. Theme Variants: Right-click the theme button or use Ctrl+Shift+V to see variants
  3. Auto Mode: Enable the auto-theme checkbox to switch based on time
  4. System Sync: The theme will automatically match your system preference if no manual selection is made
